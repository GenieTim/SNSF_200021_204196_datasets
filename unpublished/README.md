# Unpublished Results & Inputs

This folder contains data, scripts and notes that did not make the cut into publications.
Partially because the results were not promising, partially because of a lack of time.

Additionally, some simulation inputs and scripts used during experiments that later led to publications
are stored here for completeness.

In particular, this folder is currently hosting:
- [./biaxial_extension/](./biaxial_extension/): Scripts and notes on biaxial extension simulations of polymer networks.
- [./uniaxial_extension/](./uniaxial_extension/): Scripts and notes on uniaxial extension simulations of polymer networks.
