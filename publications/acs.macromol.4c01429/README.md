# 

This folder contains the data for the publication "Molecular Model for Linear Viscoelastic Properties of Entangled Polymer Networks".
Refer to the CITATION.cff file for more info on how to cite.

Here's how this folder is organized:

- in the folder [processed-data](./processed-data/), the data used for plotting in the publication can be found
- in the folder [input-scripts](./input-scripts/), the LAMMPS input templates used to generate the data can be found
